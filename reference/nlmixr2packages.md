# List all packages in the nlmixr2

List all packages in the nlmixr2

## Usage

``` r
nlmixr2packages(include_self = TRUE)
```

## Arguments

- include_self:

  Include nlmixr2 in the list?

## Examples

``` r
nlmixr2packages()
#>  [1] "cli"          "crayon"       "dplyr"        "purrr"        "rstudioapi"  
#>  [6] "nlmixr2est"   "nlmixr2extra" "rxode2"       "lotri"        "nlmixr2plot" 
#> [11] "tibble"       "magrittr"     "nlmixr2"     
```
