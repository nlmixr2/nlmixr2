# Package index

## Installation and Loading support

- [`nlmixr2CheckInstall()`](https://nlmixr2.github.io/nlmixr2/reference/nlmixr2CheckInstall.md)
  : Check your nlmixr2 installation for potential issues
- [`nlmixr2conflicts()`](https://nlmixr2.github.io/nlmixr2/reference/nlmixr2conflicts.md)
  : Conflicts between the nlmixr2 and other packages
- [`nlmixr2deps()`](https://nlmixr2.github.io/nlmixr2/reference/nlmixr2deps.md)
  : List all nlmixr2 dependencies
- [`nlmixr2packages()`](https://nlmixr2.github.io/nlmixr2/reference/nlmixr2packages.md)
  : List all packages in the nlmixr2
- [`nlmixr2update()`](https://nlmixr2.github.io/nlmixr2/reference/nlmixr2update.md)
  : Update nlmixr2 packages
